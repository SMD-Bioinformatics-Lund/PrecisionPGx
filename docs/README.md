## Installation
Clone the repository:
```
git clone https://github.com/SMD-Bioinformatics-Lund/PrecisionPGx.git .
```

View available branches:
```
git branch -a
```

Check out the relevant branch, example:
```
git checkout nf-core
```

You can find additional general documentation about installing, configuring and running nf-core pipelines on the website: [https://nf-co.re](https://nf-co.re)
