## Documentation

Documentation is built using [Sphinx](https://www.sphinx-doc.org/) and hosted on [Read the Docs](https://readthedocs.org/).

To build the docs locally, run:

```bash
cd docs
make html
```

The rendered HTML will be available in `docs/_build/html`.
